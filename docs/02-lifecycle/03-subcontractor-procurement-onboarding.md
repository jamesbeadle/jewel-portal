**Lifecycle stage 03.** Detailed workflow: [`/03-workflows/03-subcontractor-procurement-onboarding.md`](../03-workflows/03-subcontractor-procurement-onboarding.md).

## Purpose

From package issue to bid comparison to award, then onboarding, insurance / RAMS / CIS verification, work order issue, ready-to-mobilise. Pre-award compliance is a hard gate.

## Key concepts at this stage

Subcontractor, Bid Package, Quote, Work Order, Compliance Document, RAMS, CIS Status, Renewal Event.
