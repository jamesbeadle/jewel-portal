**Lifecycle stage 05.** Detailed workflow: [`/03-workflows/05-rfis-submittals-variations-delays.md`](../03-workflows/05-rfis-submittals-variations-delays.md).

## Purpose

The change layer on a live project. RFIs, submittal approvals (materials / samples / manufacturer details before installation), variations (with VO → procurement loop where pricing needs a subcontractor), notices of delay, technical clarifications.

## Key concepts at this stage

RFI, Submittal, Variation, NoD, Instruction Log.
