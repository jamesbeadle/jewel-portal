# 06 — Site Delivery, Programme & Reporting

**Lifecycle stage 06.** Detailed workflow: [`/03-workflows/06-site-delivery-programme-reporting.md`](../03-workflows/06-site-delivery-programme-reporting.md).

## Purpose

The live-site stage. Programme (Gantt tied to BoQ), daily diary, progress against work areas / packages / cost codes, photos, snags, weather / access, lookahead planning, quality observations, site actions, inspections.

## Key concepts at this stage

Programme Task, Site Report, Observation, Inspection (instance), Defect (early identification).
