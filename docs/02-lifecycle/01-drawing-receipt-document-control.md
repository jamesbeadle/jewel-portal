# 01 — Drawing Receipt & Document Control

**Lifecycle stage 01.** Detailed workflow: [`/03-workflows/01-drawing-receipt-document-control.md`](../03-workflows/01-drawing-receipt-document-control.md).

## Purpose

Once information arrives, control it properly. The system tracks what was received, what revision it is, who was issued it, what acknowledgments have come back from the field, and what RFIs / variations / instructions stem from each drawing.

## Key concepts at this stage

Drawing, Drawing Revision, Drawing Issue Record, Submittal, Correspondence / Instruction Log.
