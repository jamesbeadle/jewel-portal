**Lifecycle stage 07.** Detailed workflow: [`/03-workflows/07-valuations-cashflow-forecasting.md`](../03-workflows/07-valuations-cashflow-forecasting.md).

## Purpose

The commercial outputs. Programme Valuation Report per Claim Period (Key Output #1). Cashflow forecast (Key Output #2) built purely from project data — work-order commitments, expected valuations, predicted completion %s. Timesheet capture for cost-code allocation feeds completion % prediction.

## Key concepts at this stage

Valuation, Claim Period, Cashflow Forecast Snapshot, Timesheet, Timesheet Approval, Cost Code Budget, Cost Code Allocation.
