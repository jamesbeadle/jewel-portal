# 04 — H&S Site Mobilisation & Compliance

**Lifecycle stage 04.** Detailed workflow: [`/03-workflows/04-hs-site-mobilisation-compliance.md`](../03-workflows/04-hs-site-mobilisation-compliance.md).

## Purpose

Controlled readiness gate before site starts: welfare, signage, inductions, permits, RAMS, temporary works, fire arrangements, plant checks. Inspections / audits / incidents / corrective actions live here as the H&S engine. H&SO owns the framework; Site Manager confirms it's in place on site.

## Key concepts at this stage

Mobilisation Checklist, Inspection, Audit, Incident, Near Miss, Corrective Action, Permit, Toolbox Talk, Induction Record.
