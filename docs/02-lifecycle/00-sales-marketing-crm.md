# 00 — Sales, Marketing & CRM

**Lifecycle stage 00.** Detailed workflow: [`/03-workflows/00-sales-marketing-crm.md`](../03-workflows/00-sales-marketing-crm.md).

## Purpose

The front end of the business: lead capture → qualification → site visit → drawings received → bid/no-bid → proposal → won / lost. The won handoff creates the project shell that flows into stage 01.

## Key concepts at this stage

Lead, Opportunity, Contact, Company / Architect Practice, Site Visit, Proposal, Win/Loss Reason.
