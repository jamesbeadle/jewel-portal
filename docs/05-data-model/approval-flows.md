# Approval Flows

Who signs off what, at what threshold. Detailed per workflow; this is the at-a-glance reference.

| Decision | Default approver | Escalation above threshold |
|---|---|---|
| Bid / no-bid on a lead | PM | Director (high-value or unusual scope) |
| Tender issued to client | QS + PM | Director |
| Subcontractor award | PM | Director (high-value) |
| Compliance gate at award | OCC + H&SO (compliance pass) | — (hard gate, not threshold) |
| Mobilisation gate (move to live delivery) | Site Manager + H&SO | Director override only |
| RFI reply received from architect | PM | — |
| Submittal approval | Architect | — (architect-side) |
| Variation pricing | QS | — |
| Variation sign-off | Architect (technical) + Client (commercial) | Director (high-value internal sign-off) |
| Notice of Delay issue | PM | Director (high-value contractual impact) |
| Site report sign-off (weekly / monthly) | PM | — |
| Timesheet weekly batch approval | PM | — |
| Cost-code overrun | FD | Director (above higher threshold) |
| Programme Valuation Report issue | QS + PM | Director |
| Payment release decisions | FD | Director (above higher threshold) |
| Defect sign-off (individual) | PM (commercial) + Site Manager (physical) | — |
| Practical Completion | Architect + Client | — |
| Zero-rated VAT analysis | FD drafts | Architect / Client agree; Director sign-off |
| Retention release | FD | — (auto on settlement close + defects sign-off) |
| Portfolio-level threshold breach | FD reviews | Director escalation |

Thresholds (£ value, % deviation, ageing days) are configurable per organisation and per workflow.
