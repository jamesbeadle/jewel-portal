# Status Models

The lifecycle of every stateful entity. Confirmed per workflow during deep-dive walkthroughs.

## Lead
`New Lead → Qualified → Survey Booked → Survey Complete → Awaiting Information → Drawings Received → Feasibility/Budget Review → Tendering → Proposal Issued → Negotiation → Won | Lost | Nurture`

## Project
`Shell (on Won) → Pre-Construction → Tender → Mobilisation → Live → Close-Out → Aftercare → Closed`

## Drawing Revision
`Draft → Issued → Acknowledged on site → Superseded | Archived`

## Bid Package
`Drafting → Issued → Quotes in → Comparing → Awarded | Cancelled`

## Quote
`Invited → Open → Submitted → Withdrawn | Selected | Rejected`

## Work Order
`Drafted → Approved → Mobilising → Active → Substantially complete → Closed`

## Variation
`Drafted → Pricing → Awarded (if sub procurement loop) → Awaiting client approval → Approved | Rejected | Withdrawn`

## RFI
`Open → Awaiting reply → Replied → Closed | Implies variation`

## Submittal
`Drafted → Issued for approval → Approved | Rejected | Resubmit required`

## Mobilisation Checklist
`Open → In progress → Complete → Sign-off gate cleared`

## Inspection
`Scheduled → In progress → Complete with findings | Complete clean | Overdue → escalated`

## Observation
`Open → Investigating → Corrective action raised → Closed`

## Incident / Near Miss
`Reported → Investigating → Corrective actions in flight → Closed → Lessons learned`

## Corrective Action
`Open → In progress → Evidence submitted → Closed | Overdue → escalated`

## Timesheet Approval
`Drafting → Submitted → Approved | Rejected (back to draft)`

## Valuation
`Building (mid-period) → Draft (period end) → Approved → Issued → Acknowledged by architect/client`

## Defect
`Identified → Assigned → In progress → Evidence submitted → Signed off | Disputed → reopened`

## Settlement Record
`Open at PC → Open items being settled → VAT analysis drafted → VAT analysis issued → Client agreement → Final settlement → Retention released → Project closed`
